üîß How to Use in Your Application

When a user clicks on "Print Bill" in the billing interface, the thermal print function is called:

// Import the function
import { printThermalBill } from '../../utils/exportUtils';

// Call it with bill data
const handlePrintBill = (bill) => {
  printThermalBill(bill);
};

// Example bill object:
const bill = {
  billNo: 145,
  billDate: '2025-11-18T15:45:00',
  customerName: 'Rahul Kumar',
  customerMobile: '9123456789',
